# 基于图像的信息隐藏技术研究 python1007

## 项目概述

本项目是基于Python技术实现的图像信息隐藏研究，旨在构建一个能够上传图片、添加隐藏信息以及解锁隐藏信息的网站平台。

## 技术栈

- **后端语言**：Python 3.7
- **数据库**：MySQL
- **编译器**：PyCharm
- **前端技术**：未明确说明

## 功能模块

本项目包含以下主要功能模块：

- 图片上传
- 隐藏信息添加
- 隐藏信息解锁
- 数据存储与检索

### 核心算法

项目使用最低有效位（LSB）算法进行信息嵌入，实现了图像的深入处理。
